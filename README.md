
#Dollars to cents converter
Dollars to cents converter
Program to convert dollar to cents with the cents of:

Quarters (25 ¢)
Dimes (10 ¢)
Nickels (5 ¢)
Pennies (1 ¢)
How to use
Clone the repository 


Open the Dollars-to-cents-converter folder
Run the index.html file
